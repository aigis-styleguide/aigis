コマンドラインから利用する

インストール

npm i node-aigis

バージョンの確認とヘルプの表示

バージョンを表示する

$ ./node_modules/.bin/aigis --version
1.0.0

ヘルプを表示する

$ ./node_modules/.bin/aigis --help
Usage: aigis run -c ./path/to/aigis_config.yml

Commands:
  run   Generate styleguide
  init  Copy template & theme & config file

Options:
  -v, --version  Show version number                                   [boolean]
  -h, --help     Show help                                             [boolean]

Examples:
  node_modules/.bin/aigis run -c ./config_file.yml
  node_modules/.bin/aigis init -e jade

aigisを実行する

./node_modules/.bin/aigis run

オプション

name alias value
--config -c ./path/to/aigis_config.yml

設定ファイルとテンプレートを作成する

./node_modules/.bin/aigis init

オプション

name alias value
--engine -c ejs, jade, hbs

gulpで利用する

インストール

$ npm install --save-dev gulp-aigis

gulp-aigisを実行する

var gulp = require('gulp');
var aigis = require('gulp-aigis');

gulp.task('aigis', function() {
  return gulp.src('./path/to/aigis_config.yml')
    .pipe(aigis());
});

Node.jsで利用する

インストール

npm install node-aigis

実行

var Aigis = require('node-aigis');
var aigis = new Aigis('./path/to/aigis_config.yml');
aigis.run();
